Instalação Simples

Instale o Kafta rapidamente usando Go. Requer Go 1.18+ e acesso a um cluster Apache Kafka.

# Instalar a versão mais recente
go install github.com/electric-saw/kafta/cmd/kafta@latest
# Verificar instalação
kafta --help
# Verificar versão
kafta version
# Para Go < 1.18
go get -u github.com/electric-saw/kafta
# Compilar da fonte
git clone https://github.com/electric-saw/kafta.git
cd kafta
go build -o kafta cmd/kafta/main.go
# Configurar PATH (se necessário)
export PATH=$PATH:$(go env GOPATH)/bin
# Configurar cluster Kafka
kafta config set-cluster my-cluster --bootstrap-servers localhost:9092
# Ativar auto-complete (bash)
kafta completion bash >> ~/.bashrc

Exemplos de Uso

Veja como usar o Kafta para gerenciar seus clusters Kafka de forma eficiente.

Listar Tópicos
Visualize todos os tópicos do cluster
$ kafta topic list
✓ user-events
✓ order-events
✓ payment-events
Criar Tópico
Crie novos tópicos com configurações específicas
$ kafta topic create my-topic --partitions 3 --replication-factor 1
✓ Topic 'my-topic' created successfully
Consumer Lag
Monitore o lag dos consumer groups
$ kafta consumer lag my-group
TOPIC PARTITION LAG
my-topic 0 15
my-topic 1 3
Produzir Mensagens
Envie mensagens para tópicos interativamente
$ kafta producer --topic my-topic
> Hello Kafka!
✓ Message sent to partition 0, offset 1234
Schema Registry
Gerencie schemas Avro
$ kafta schema subjects-list
user-value
order-key
payment-value
Consumir Mensagens
Consuma mensagens em tempo real
$ kafta console consumer --topic my-topic --group my-group
{"id": 1, "message": "Hello Kafka!"}
{"id": 2, "message": "Another message"}
