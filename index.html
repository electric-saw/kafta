Instalação Simples

Escolha seu sistema operacional e siga as instruções para começar a usar o Kafta em minutos.

# Instalar via Homebrew
brew install kafta
# Ou via curl
curl -fsSL https://install.kafta.dev | sh
# Configurar shell
echo 'eval "$(kafta init)"' >> ~/.zshrc
source ~/.zshrc
# Ubuntu/Debian
wget -qO- https://install.kafta.dev/deb | sudo bash
# Fedora/CentOS/RHEL
curl -fsSL https://install.kafta.dev/rpm | sudo bash
# Instalação universal
curl -fsSL https://install.kafta.dev | sh
# Configurar shell
echo 'eval "$(kafta init)"' >> ~/.bashrc
source ~/.bashrc
# PowerShell (Recomendado)
iwr -useb https://install.kafta.dev/windows | iex
# Ou via Chocolatey
choco install kafta
# Ou via Scoop
scoop bucket add extras
scoop install kafta
# Configurar PowerShell
Add-Content $PROFILE 'Invoke-Expression (kafta init)'

Exemplos Práticos

Veja como o Kafta transforma comandos complexos em linguagem natural e vice-versa.

Buscar Arquivos
Encontre arquivos por nome ou conteúdo
💬 "encontrar todos os arquivos .js modificados hoje"
⚡ find . -name "*.js" -mtime 0
Gerenciar Processos
Controle processos do sistema facilmente
💬 "mostrar processos que usam mais CPU"
⚡ ps aux --sort=-%cpu | head -10
Git Operations
Simplifique comandos Git complexos
💬 "fazer commit de todos os arquivos com mensagem 'fix: correção de bugs'"
⚡ git add -A && git commit -m "fix: correção de bugs"
Compressão de Arquivos
Arquive e comprima arquivos facilmente
💬 "comprimir pasta 'projeto' em arquivo zip"
⚡ zip -r projeto.zip projeto/
Monitoramento de Sistema
Verifique status e recursos do sistema
💬 "mostrar uso de disco por diretório"
⚡ du -h --max-depth=1 | sort -hr
Rede e Conectividade
Teste conexões e diagnósticos de rede
💬 "testar conectividade com google.com"
⚡ ping -c 4 google.com
