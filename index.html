Simple Installation

Install Kafta quickly using Go. Requires Go 1.18+ and access to an Apache Kafka cluster.

# Install latest version
go install github.com/electric-saw/kafta/cmd/kafta@latest
# Verify installation
kafta --help
# Check version
kafta version
# For Go < 1.18
go get -u github.com/electric-saw/kafta
# Build from source
git clone https://github.com/electric-saw/kafta.git
cd kafta
go build -o kafta cmd/kafta/main.go
# Configure PATH (if needed)
export PATH=$PATH:$(go env GOPATH)/bin
# Configure Kafka cluster
kafta config set-cluster my-cluster --bootstrap-servers localhost:9092
# Enable auto-complete (bash)
kafta completion bash >> ~/.bashrc

Usage Examples

See how to use Kafta to manage your Kafka clusters efficiently.

List Topics
View all topics in the cluster
$ kafta topic list
✓ user-events ✓ order-events ✓ payment-events
Create Topic
Create new topics with specific configurations
$ kafta topic create my-topic --partitions 3 --replication-factor 1
✓ Topic 'my-topic' created successfully
Consumer Lag
Monitor consumer group lag
$ kafta consumer lag my-group
TOPIC PARTITION LAG my-topic 0 15 my-topic 1 3
Produce Messages
Send messages to topics interactively
$ kafta producer --topic my-topic
> Hello Kafka! ✓ Message sent to partition 0, offset 1234
Schema Registry
Manage Avro schemas
$ kafta schema subjects-list
user-value order-key payment-value
Consume Messages
Consume messages in real-time
$ kafta console consumer --topic my-topic --group my-group
{"id": 1, "message": "Hello Kafka!"} {"id": 2, "message": "Another message"}
